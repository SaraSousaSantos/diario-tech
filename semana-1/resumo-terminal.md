# Terminal

Hostname Nome do computador

whoami Mostra quem é o utilizador atual que está logado na sessão

pwd Mostra o caminho da pasta atual

ls Lista ficheiros da pasta em que estamos

ls –R Lista ficheiros e caminho

dir Lista ficheiros da pasta em que estamos

cd nome-da-pasta Entra na pasta (caminho absoluto) EX: cd C:\Users\<teu_utilizador>\Desktop

cd .. Sobe uma pasta

cd ../../ para subir dois, e assim por diante

cd ~ diretório pessoal do utilizador

type nome-do-ficheiro Mostra o conteúdo de um ficheiro

cat nome-do-ficheiro Mostra o conteúdo de um ficheiro no terminal (Linux/macOS)

mkdir nome-da-pasta Cria uma pasta nova.

new-item nome-do-ficheiro Cria um ficheiro

touch nome-do-ficheiro Cria um ficheiro (Linux/macOS)

mv nome-do-ficheiro destino Move ficheiros/pastas

mv nome-do-ficheiro novo-nome Renomear ficheiros/pastas

Para mover o ficheiro de folder-1 para folder-2, estando dentro de folder-1:

mv index.txt ../folder-2

cp nome-do-ficheiro destino Copia ficheiros/pastas.

cp -r nome-da-pasta destino Copiar pastas com conteúdo

rm nome-do-ficheiro Apaga o ficheiro

rm -r nome-da-pasta Apaga a pasta

rm -rf nome-da-pasta Apaga a pasta e tudo o que está dentro dela

echo "Olá terminal!" > ola.txt Cria um ficheiro ola.txt e escreve “Olá terminal!” nele. Se o ficheiro já existir, substitui o conteúdo
